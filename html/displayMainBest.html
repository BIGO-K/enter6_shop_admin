도움말
도움말
  • 메인페이지에 노출되는 항목을 관리할 수 있는 페이지입니다.
  • 상단바 배너는 PC전용이며 각 1개씩만 프론트에 노출이 가능합니다. (좌측 배너 먼저 등록을 해야 하며 한 곳만 등록 시 등록하지 않은 영역은 빈 영역으로 노출됨)
  • 구름/둥둥 배너는 PC전용이며 둥둥 배너는 1개만 프론트에 노출이 가능합니다.
  • 인터랙션 커튼 배너는 PC전용이며 1개만 프론트에 노출이 가능합니다.
  • GNB 카테고리 배너는 PC전용이며 카테고리별 1개씩만 프론트에 노출이 가능합니다.
  • 메인 배너는 문구 입력이 가능하며 배너 1개당 3개의 연관 추천상품 등록이 가능합니다.
  • 메인 띠배너는 상단, 중단, 하단 3영역에 노출이 가능합니다.
  • 카테고리 인기상품 배너는 문구 입력이 가능하며 PC와 Mobile 노출되는 형태가 상이합니다.
  • 스윗딜 상품 관리에서는 1세트에 4개 상품이 등록되어야 하며 상품별 문구를 지정할 수 있습니다.
  • 베스트 상품 관리에서는 최소 8상품부터 최대 24상품까지 노출이 가능하며 8상품 단위로 등록이 가능합니다.
  • 엔터식스 NOW 상품 관리에서는 8개의 상품을 필수로 등록해야 합니다.
  • 모든 변경 내역은 변경한 익일 오전에 반영됩니다.
