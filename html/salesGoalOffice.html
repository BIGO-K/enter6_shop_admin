  • 샘플파일을 다운로드하여 해당 파일의 양식에 맞게 데이터 등록
  • 샘플파일 다운로드 시, 매출목표 지점에 노출되고 있는 지점을 반영하여 엑셀파일이 다운로드됨
파일 업로드
  • 파일명: target_figure_branch.xls / 파일명은 반드시 형식에 맞게 작성 후 업로드
  • 데이터 등록은 엑셀 업로드만 가능/ 수정이 필요할 시, 엑셀을 수정하여 재 업로드
  • 데이터를 입력하지 않은 빈 영역은 0 으로 목표 금액이 자동 설정
  • 31일이 마지막 일이 아닌 월은 해당 월에 있는 일자까지만 매출목표 금액 입력
  • 지점 추가 및 수정 후 목표 업로드 시, 샘플파일을 다시 다운로드 받아 목표를 등록
