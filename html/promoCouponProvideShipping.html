배송비 쿠폰 발행 관리

도움말
도움말
  • 배송비 쿠폰을 발행할 수 있는 페이지입니다.
  • 표시된 항목은 필수입력 항목입니다.
  • 쿠폰 사용범위 탭에서 전체 상품으로 선택할 경우 판매 중인 모든 상품에 적용됩니다.
  • 쿠폰 사용범위 탭에서 특정 브랜드 상품으로 선택할 경우 선택한 브랜드의 상품에만 적용됩니다.
  • 쿠폰 사용범위 탭에서 특정 셀러 상품으로 선택할 경우 설정한 셀러가 판매하고 있는 상품에만 적용됩니다.
  • 쿠폰 사용범위 탭에서 특정 상품으로 선택할 경우 설정한 상품에만 적용됩니다.
  • 쿠폰 사용범위 탭에서 지점 상품으로 선택할 경우 설정한 엔터식스 지점의 상품에만 적용됩니다.
  • 사용 여부 탭에서 사용으로 선택할 경우 프론트에서 주문 시 사용 가능합니다.
  • 사용 여부 탭에서 미사용으로 선택할 경우 프론트에서 주문 시 사용이 불가합니다.
  • CS 쿠폰 여부 체크 시 CS에서 발행하는 쿠폰으로 표시되며 미체크 시 일반 관리자가 발행하는 쿠폰으로 표시됩니다.
