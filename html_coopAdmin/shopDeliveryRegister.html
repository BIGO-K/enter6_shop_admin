무료배송 설정
  • 사용 설정 시 판매중인 모든 상품 및 등록되는 모든 상품의 기본 배송정책이 '무료배송'으로 설정
  • 무료배송 사용으로 설정이 되어 있어도 각 상품별 배송 정책은 수정 가능

이상 구매 시 무료배송
  1. 구매금액은 최종 결제금액을 기준으로 조건별 배송비를 산정
  2. 기본 배송료를 입력하지 않으면 배송료는 '0원'으로 처리되어 무료배송으로 설정
  3. 조건별 배송료를 입력하지 않으면 배송료는 기본 배송비를 기준으로 산정
묶음배송 설정
  • 묶음배송 사용 설정 시 동일 판매점의 상품을 여러 개 구매하면 1번의 배송비만 산정되어 적용
  • 묶음상품 중 무료배송 상품이 존재할 시 무료배송으로 처리

무료배송 상품 반품배송비
  1. 구매자 귀책 시 반품 배송비가 적용되며, 판매자 귀책 시 반품 배송비 적용 안됨
  2. 상품 구매 시 배송비를 결제한 반품 요청 주문 건은 왕복 비용 적용
  3. 상품 구매 시 무료배송일 경우 설정한 값에 따라 반품 배송비의 2배 또는 반품 배송비 적용

  2. 상품 구매 시 배송비를 결제한 교환요청 주문 건은 왕복 비용 적용
  3. 상품 구매 시 무료배송일 경우 설정한 값에 따라 교환 배송비의 2배가 적용
